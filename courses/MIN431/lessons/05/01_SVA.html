1. Simulación de variables aleatorias reales

Nuestro objetivo será generar números que puedan ser considerados como realizaciones de una variable aleatoria con una distribución de probabilidad dada.

Spoiler!! No existe una forma de generar números “al azar”.

Del punto de vista computacional, todos los métodos de simulación de números “al azar” generan secuencias periódicas. Por lo tanto, se contenta con generar una sucesión de números “seudo-aleatorios”, cuyo período es suficientemente largo para que presenten, para el usuario, todas las características de números al azar.

1.1. Simulación de una variable uniforme en el intervalo \([0,1]\)

La densidad de probabilidad de una variable aleatoria \(U\) uniforme en [0,1] es:

\begin{equation} f(u) = \begin{cases} 1 & \text{si } 0 < u < 1 \\ 0 & \text{en caso contrario} \end{cases} \end{equation}

import numpy as np
import plotly.graph_objects as go

# densidad de probabilidad teórica
x = np.linspace(-1, 2, 500) # Dominio de la función
pdf = np.where((x >= 0) & (x <= 1), 1 , 0)

# PDF line
pdf_line = go.Scatter(
    x=x,
    y=pdf,
    mode='lines',
    name='Densidad de probabilidad teórica',
    line=dict(color='blue', width=2)
)

# Combine in a figure
fig = go.Figure(pdf_line)
fig.update_layout(
    title="Distribución uniforme en [0, 1]",
    xaxis_title="Valor",
    yaxis_title="Densidad de probabilidad",
    bargap=0.05
)

fig.show()
