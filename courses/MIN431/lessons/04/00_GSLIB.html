1. Instalando la GSLIB

GSLIB (Geostatistical Software Library) es una de las primeras colecciones de software geoestadístico del mundo. Fue desarrollada por estudiantes y profesores de la Universidad de Stanford, con Clayton Deutsch y André Journel como autores principales.

Características clave

  • Finalidad: GSLIB se utiliza para una variedad de tareas geoestadísticas, como el análisis de datos, la estimación de variogramas, el kriging y las simulaciones estocásticas en 1D, 2D y 3D.

  • Código original: El código original de la biblioteca está escrito en Fortran, pero sus algoritmos han sido ampliamente utilizados y reimplementados en otros lenguajes.

  • Dominio público: GSLIB es de dominio público y se distribuye con el código fuente y ejecutables para PC.

  • Reimplementación en Python: Los algoritmos de GSLIB se han reimplementado en Python en bibliotecas como GeostatPy, lo que facilita su uso en entornos de programación modernos.

  • Aplicaciones: Es una herramienta fundamental para geólogos, ingenieros de minas y científicos de datos en el análisis de datos espaciales y la estimación de recursos.

1.1.1. Descargar Programas

[ ]:

%%bash

url="http://www.statios.com/software/gslib90_ls.tar.gz"
wget -O gslib90_ls.tar.gz $url
tar -xf gslib90_ls.tar.gz
rm gslib90_ls.tar.gz

mkdir Gslib90

gfortran -w -O3 -fallow-argument-mismatch -o ./Gslib90/gamv ./gslib90/gamv.for ./gslib90/gslib/*.for
gfortran -w -O3 -fallow-argument-mismatch -o ./Gslib90/vmodel ./gslib90/vmodel.for ./gslib90/gslib/*.for
gfortran -w -O3 -fallow-argument-mismatch -o ./Gslib90/ik3d ./gslib90/ik3d.for ./gslib90/gslib/*.for
gfortran -w -O3 -fallow-argument-mismatch -o ./Gslib90/kt3d ./gslib90/kt3d.for ./gslib90/gslib/*.for

rm -r gslib90

1.2.1. Descargar Programas

Puedes descargar directamente de la pagina

http://www.statios.com/software/gslib90.zip

y descomprimir los programas.

Asegúrate de que todos se puedan acceder dentro de la misma carpeta Gslib90.

También puedes usar el siguiente código en Jupyter.

[ ]:

import urllib.request
import zipfile
import os

url = "http://www.statios.com/software/gslib90.zip"
zip_path = "gslib90.zip"

# Descargar el archivo zip
urllib.request.urlretrieve(url, zip_path)

# Descomprimir el zip en el directorio actual
with zipfile.ZipFile(zip_path, "r") as z:
    z.extractall()

# Eliminar el archivo zip descargado
os.remove(zip_path)
